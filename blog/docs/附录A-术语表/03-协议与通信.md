title: 协议与通信
---

- **JSON-RPC** — 一种基于 JSON 的远程调用协议，KQode 的 TUI 前端与 Rust 后端之间用它通信。
- **transport** — 传输层，承载协议消息的通道（例如基于 stdin/stdout 的管道），与「消息格式」解耦。
- **request** / **response** / **notification** — JSON-RPC 的三类消息：请求（需要回复）、响应（对请求的回复）、通知（单向、不需要回复）。
- **ACK**（acknowledgment，确认）— 后端收到并受理某条消息后，回给前端的确认回执。
- **stdin** / **stdout** — 标准输入 / 标准输出，进程间传递数据的默认通道。
