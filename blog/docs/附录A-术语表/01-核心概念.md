title: 核心概念
---

- **Coding Agent** — 能理解需求、读写代码、执行命令并自主推进编程任务的 AI 代理程序。KQode 本身就是一个 Coding Agent，对标 Codex、Claude Code、Copilot CLI。
- **Agent Harness** / **Harness** — 把「模型」包成一个可用产品的整套外围工程：agent loop、工具调用、文件与命令执行、会话、审批、失败恢复等。模型是引擎，Harness 是让引擎真正能干活的底盘。中文常译作「框架」或「脚手架」，但都不够贴切，故保留英文。
- **Agent Runtime** — Agent 运行时，指驱动 agent loop 持续运转的核心运行环境，包含模型调用、工具注册、状态流转等。
- **Harness Engineering** — 围绕 Agent Harness 的工程实践与方法论，也是本项目想学习和研究的主题。
- **agent loop** — Agent 的主循环：接收输入 → 调用模型 → 执行工具 → 观察结果 → 再决策，如此往复直到任务完成。
