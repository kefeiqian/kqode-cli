- **xtask** — Rust 生态里「用 Rust 写项目自动化脚本」的约定；KQode 通过 `cargo xtask <命令>` 跑构建、测试、blog 等任务。
- **commit** / **tag** — Git 的提交 / 标签；本博客每个 U 单元对应一个打了 tag 的 commit。
- **frontmatter** — Markdown 文件顶部用 `---` 包裹的元数据（如 `title`、`sidebar_position`）。
- **locale** — 语言区域；本博客有 `zh-Hans` 与 `en` 两个 locale。
- **baseUrl** — 站点部署的基础路径，本博客是 `/kqode-cli/`。
- **OIDC**（OpenID Connect）— 一种身份认证协议；npm 的 Trusted Publishing 借助它实现免长期令牌发布。
- **Trusted Publishing** — npm 的可信发布机制，用 CI 的 OIDC 身份代替长期 token 来发布包。
- **optionalDependencies** — `package.json` 里的可选依赖；KQode 用它按平台分发二进制。
