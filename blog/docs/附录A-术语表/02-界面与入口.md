title: 界面与入口
---

- **CLI**（Command-Line Interface，命令行界面）— 在终端里通过命令使用的程序形态。
- **TUI**（Terminal User Interface，终端用户界面）— 在终端里渲染的富交互界面，不只是逐行文本输出。KQode 的 TUI 用 Ink 构建。
- **Ink** — 用 React 组件编写终端 UI 的框架，KQode 的 TUI 基于它。
- **composer** — TUI 里输入 prompt 的编辑区（输入框），负责多行编辑、光标、提交等交互。
- **headless** — 无界面运行模式：核心逻辑不依赖 TUI 也能独立运行，供 CLI 与自动化使用。中文「无头」不够直观，保留英文。
- **daemon** — 常驻后台的服务进程，是 KQode 规划中的守护进程模式。
