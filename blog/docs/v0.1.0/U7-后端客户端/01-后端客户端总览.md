## 核心路径

```text
startBackendRuntime(store, client)
  -> client.ensureStarted()
  -> createBackendClient(...)
  -> launchSourceBackend(...)         // U6
  -> openReadyConnection(...)
  -> waitForBackendReady(...)
  -> createMessageConnectionClient(connection)
  -> submitMessage({ text })
  -> connection.sendRequest(messageSubmitRequest, params)
```

最小的 client wrapper 如下：

```ts
export function createMessageConnectionClient(connection: MessageConnection): BackendClient {
  return {
    async submitMessage(params: MessageSubmitParams): Promise<MessageSubmitResult> {
      try {
        return await connection.sendRequest(messageSubmitRequest, params);
      } catch (error) {
        throw toBackendClientError(error);
      }
    }
  };
}
```

这个函数体现了 U7 的设计目标：UI 不需要知道 `Content-Length` header、request id 或 `MessageConnection`，只需要调用 `submitMessage`。
